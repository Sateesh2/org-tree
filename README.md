# Org Tree UI Library

A React component library for rendering organizational tree structures.

## 🛠️ Status

**This project is in early development.** Core components and features are being built incrementally.

## 📌 Goals

- Render hierarchical organizational trees
- Customizable node UI
- Expand/collapse functionality
- Drag-and-drop support (optional)
- Keyboard accessibility and responsiveness

## 🚀 Getting Started

```bash
# Clone the repository
git clone https://github.com/your-org/org-tree-ui.git
cd org-tree-ui

# Install dependencies
npm install
# or
yarn install

# Start the development server
npm run dev
